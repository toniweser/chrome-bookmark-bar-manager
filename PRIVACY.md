Bookmark Bar Manager does not collect, store, or transmit any personal data.

All bookmark data is stored locally in your browser using the Chrome Bookmarks API. The only value persisted is the ID of your currently active bookmark set, stored locally via the Chrome Storage API.

No data is sent to external servers. No analytics, tracking, or third-party services are used.
